# Mobile experience

Chive works on mobile browsers with layouts optimized for smaller screens.

## Navigation

On mobile, the main navigation collapses into a hamburger menu in the top-left corner. Tap it to access:

- Home
- Authors
- Browse
- Trending
- Search
- Dashboard (when signed in)

## Search

The search bar is accessible from the hamburger menu or from the search icon in the header. Search works the same as on desktop: enter keywords, use field prefixes, and apply facets. The facet panel opens as a full-screen overlay on mobile.

## Eprint pages

Eprint pages stack vertically on mobile:

- Title and metadata appear at the top
- The abstract follows below
- Tabs for reviews, citations, and related works are swipeable
- The PDF viewer adapts to screen width

## Author cards

Author discovery cards on the authors page and browse page use a single-column layout on narrow screens. Each card shows the author's name, handle, avatar, and a mute button.

## Dashboard

The dashboard sidebar collapses into a dropdown menu on mobile. All dashboard features (submissions, settings, notifications) remain accessible.

## Tips

1. Use landscape orientation for reading PDFs
2. Tap column headers in tables to sort
3. Pull down to refresh on most pages
4. Long-press links to open in a new tab
